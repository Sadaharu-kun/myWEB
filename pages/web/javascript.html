Variables

  • let : is flexible (block-scoped)
  • const : is not flexible (block-scoped)
  • var : is flexible, but since its accessable throughout the entire function where its declared, it can lead to unexpected behaviours and bugs if used within blocks like loops or if-statements (function scoped!!)

    • Data types

    Primitive:
  • Number

    Represents both integers and decimals

  • String

    Represents a sequence of characters

  • Boolean

    Represents true or false values

  • Undefined

    Represents an uninitialised variable

  • Null

    Represents an intentional absence of any object value

    • Operators

    Arithmetic:
  • +
  • -
  • *
  • /
  • % modulus/remainder
  • ** exponentation

    • example: 

    
    let product = 4 * 2;    // = 8
    let quotient = 18 / 2;  // = 9
    
    let remainder = 10 % 3; 
    // = 1 (10/3 quotient is 3,
    and remainder is 1 [10 = 3*3 + 1])
    
    let power = 2 ** 3;     // (2*2*2) = 8

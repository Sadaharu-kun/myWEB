Flexbox:

  1. Start with a container, like div. And give it the property flex ('display: flex;')

    2. flex-container has control over how it's children are layed out

    Changing the item width to 100% (the div container) the flexible container displays items evenly across main axis. by targeting items flex property (.purple-item { flex: 2; } now takes up 50% instead of 1/3 )

    x axis: (justify-content: flex-start/-end/center/space-evenly/around)

    y axis: (align items) + (justify-content: '-:-')

    flex-direction: column -> x and y axis switches place! (works the same but tilt head sideways)

display: flex;
flex-wrap: nowrap; // items inside container
// : wrap // items placed out. If not fit? One step down
